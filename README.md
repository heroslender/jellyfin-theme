# jellyfin-theme
My personal custom theme for jellyfin to be used with [ElegantFin](https://github.com/lscambo13/ElegantFin).

Things this changes:
  - Makes the [Media Bar](https://github.com/IAmParadox27/jellyfin-plugin-media-bar) appear with the full width;
  - Removes the video format and audio codec labels from [Jellyfin Enhanced](https://github.com/n00bcodr/Jellyfin-Enhanced);
  - Places the watch progress bar at the bottom of the card.
